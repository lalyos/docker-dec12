
## Common Comkmands

- ls - list files
  - -l long
  - -a hidden
  - -t sort by time
  - -r reverse 
- mkdir - create dir
- cd - navigate dir
- rm - delete file
- echo - prints to stdout
- cat 
  - cat file - reads from file prints to STDOUT
  - cat > file - writes to file
  - cat > file <<EOF - HERE-DOC ...
